# TowerUP
You can visit this website **[here](https://its-idhem.github.io/TowerUp/)**. I essentially tried my best to replicate SandTrix

## About Game
This game is like regular tetris, but the blocks have sand physics. Get a solid colour object to touch both walls of the game area to get a clear!
**Controls:**
- Left and right arrow keys for moving blocks left and right respectivelly
- Up arrow to spin block
- Down arrow to make block fall faster
- Space bar to make a block fall instatelly

Have fun!!
